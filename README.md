# Major Historical Events of Indigenous Peoples in Taiwan - Interactive Series

[![GitHub Pages](https://img.shields.io/badge/Deployment-GitHub%20Pages-blue?style=for-the-badge)](https://ss1111119.github.io/indigenous-history-taiwan/)

## 專案簡介 | Project Overview

本專案致力於將原住民族委員會出版之《原住民族重大歷史事件系列叢書》轉化為互動式網頁體驗。透過數位敘事，我們希望讓大眾更直觀地了解台灣原住民族在歷史長河中的抗爭、遷徙與家園守護的故事。

This project transforms the "Major Historical Events of Indigenous Peoples" book series (published by the Council of Indigenous Peoples) into an interactive web experience. Through digital storytelling, we aim to provide an intuitive understanding of the resistance, migration, and ancestral land protection of Taiwan's Indigenous peoples.
